# AI Chat

AIと気軽に会話できるチャットサイトです。
ニックネームを設定して、ブラウザ上でAIとメッセージのやり取りができます。

PC・スマホのどちらでも使えます。

```text
ユーザー → メッセージ送信 → AIが返答
```

## このサイトでできること

* AIとチャットできる
* ニックネームを設定できる
* チャット履歴を保存できる
* ページを閉じても、同じ端末・同じブラウザなら履歴が残る
* 履歴を削除できる
* Enterキーで送信できる
* Shift + Enterで改行できる
* スマホでもPCでも見やすい画面で使える

## 使い方

1. サイトを開きます
2. ニックネームを入力します
3. メッセージ欄に話したいことを書きます
4. 送信ボタンを押します
5. AIから返事が返ってきます

Enterキーでも送信できます。
文章を改行したいときは、Shift + Enterを使ってください。

## チャット履歴について

チャット履歴は、使っている端末のブラウザ内に保存されます。

そのため、次の場合は履歴が引き継がれません。

| 場合             | 履歴          |
| -------------- | ----------- |
| 同じ端末・同じブラウザで開く | 残ります        |
| 別のスマホやPCで開く    | 引き継がれません    |
| 別のブラウザで開く      | 引き継がれません    |
| シークレットモードで開く   | 残らないことがあります |
| ブラウザのデータを削除する  | 消えます        |

履歴を消したい場合は、画面上の「履歴を消す」ボタンを押してください。

## 入力しないほうがいい情報

AIとの会話には、個人情報や大事な情報を入力しないようにしてください。

入力しないほうがいいものの例：

* 本名
* 住所
* 電話番号
* メールアドレス
* パスワード
* クレジットカード情報
* 学校や家が特定できる内容
* 他人に見られたら困る内容

## 注意事項

AIの返答は、必ず正しいとは限りません。
勉強、調べもの、文章作成などに使う場合も、最終的には自分で確認してください。

特に、次のような内容はAIの返答をそのまま信じすぎないようにしてください。

* テストや課題の答え
* 重要な予定や手続き
* お金に関すること
* 健康に関すること
* 法律や契約に関すること

## うまく動かないとき

### AIから返事が来ない

少し待ってから、もう一度送信してみてください。
通信状況や混雑によって、返事に時間がかかることがあります。

### エラーが表示される

ページを再読み込みしてから、もう一度試してください。
それでも直らない場合は、時間を空けてから試してください。

### 画面が崩れている

ページを再読み込みしてください。
それでも直らない場合は、別のブラウザで開いてみてください。

### 履歴が消えた

履歴はブラウザ内に保存されています。
シークレットモードを使ったり、ブラウザのデータを削除したりすると履歴が消えることがあります。

## おすすめの使い方

このサイトは、次のような用途に使えます。

* わからないことを質問する
* 勉強のサポートに使う
* 文章を考える
* アイデアを出す
* 雑談する
* 簡単な相談をする
